# Security Policy

## Supported Versions

Security fixes are applied to the latest released version of this repository
(Rust crate `goosefs-sdk` and Python package `goosefs`).

## Reporting a Vulnerability

Please report suspected security vulnerabilities **privately**. Do not open a
public GitHub issue or pull request for security reports.

Preferred channels (pick one):

1. [GitHub Security Advisories](https://github.com/Tencent/tencent-goosefs-rust-sdk/security/advisories/new)
   for this repository
2. Email: `datalake@tencent.com` (subject prefix: `[SECURITY] goosefs-sdk`)

Include enough detail to reproduce the issue (affected version, environment,
and a minimal PoC when possible). We will acknowledge receipt and coordinate a
fix and disclosure timeline with you.
